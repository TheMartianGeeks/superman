# Superman
Manage your app versioning by `CLI` with `superman`

## Installation
    $ npm install -g superman

## Auto update
    $ superman version:update
    
## Set version
    $ superman version:set 0.1.3
    
## Init version
    $ superman version:init

## Tests
  `npm test`
